QUINTAL DO ZÉ - SISTEMA LOCAL PROFISSIONAL

COMO RODAR:
1. Extraia este ZIP em uma pasta nova.
2. Abra a pasta no VS Code.
3. No terminal, rode:
   npm install
   npm start
4. Acesse no computador:
   http://localhost:3000
5. Para celular/tablet, use o endereço que aparece no terminal, exemplo:
   http://192.168.0.105:3000
